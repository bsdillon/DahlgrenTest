
# JavaFX GUI

This directory holds a Java FX GUI which may be compiled using `gradle` or the
wrapper scripts `./gradle` or `.\gradle.bat`.

# Run a development build

```bash
gradle run
.\\gradlew.bat run
./gradlew run
```

# Build a .jar file

```bash
gradle jar
.\\gradlew.bat jar
./gradlew jar
```

The jarfile is produced in `build/libs/JavaFXGui.jar`.
