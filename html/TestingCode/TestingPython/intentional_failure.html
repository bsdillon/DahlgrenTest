Creating a Test that Intentionally Fails

  • Start by creating a test file like in the basic test tutorial..
  • The way that you want to structure your test is so that the expected result is not what what your actual result will be. From the example in the basic test tutorial, we set up x and y and then set the expected result to something other than x + y. Then, instead of having '==' for equality, we put '!='so that we get a positive result for the intentionally incorrect expected result.
