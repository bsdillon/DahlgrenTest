Parse from PDF files

PyPDF2

pip install PyPDF2

PyPDF2 can be useful for:

  • Extracting document information
  • Splittings page by page
  • Merging documents page by page
  • Cropping pages
  • Merging multiple pages into a single page

PyPDF2 library documentation can be found here.

tabula-py

pip install tabula-py

tabula-py can read tables from PDF files. It could also convert the table from the PDF file into CSV, TSV, and JSON files. Note that it ignores all text from the PDF. It only extracts data from tables. An example of how it use this library can be found here.

Parse from WORD files

Document

pip install python-docx

Document is a library that can be utilize to create, update, and extract data from Microsoft Word files. It's built down into creating objects that represents paragraphs or text of the document, tables, sections, and more. Documentation and API of this library are found here.

docx2txt

pip install docx2txt

This library simply extracts all text from a word document. Note it does not extract tables from MS Word. Text is extracted using the docx2txt.process("file.docx")syntax. You could also write images from Word to a directory using docx2txt.process("file.txt, "/dir").

Parse from HTML files

BeautifulSoup

pip install beautifulsoup4

BeautifulSoup is a Python library that parses HTML and XML documents. It is capable of parsing text and also extract data from the table of the HTML site. An example of extracting a table and its data can be found here.
