Downloading the Virtual Machine

These instructions will take you through the process of downloading and installing VirtualBox as well as setting up your first VM. There are some changes:

  • On slide 6, select Ubuntu (64-bit) instead of Red Hat (64-bit).
  • On slide 9 I recommend a dynamically allocated disk space. This will allow you to add more space to the existing VM if you need it.
  • After slide 11 your VM is done

Setting up the virtual machine with Ubuntu

  1. Download the Ubuntu Desktop from this link. We prefer version 20.04. It's best to save this somewhere safe on your computer and not just leave it in your downloads folder.
  2. Once it downloads, start the new VM and this Ubuntu image should be automatically selected. If not you can browse for it.
  3. Choose the language. Choose to install Ubuntu rather than a trial version or something without graphics. This may take a while.
