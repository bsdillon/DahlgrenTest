Getting table values in WORD

The code snippet below takes data from tables of each cells and outputs it in a text file.

Code

1. from docx import Document
2. out = open('output.txt', 'w')
3. doc = Document('Self Critique Sp 3.docx')
4. table = doc.tables[0]
5. for row in table.rows:
6.    for cell in row.cells:
7.       for paragraph in cell.paragraphs:
8.          out.write(paragraph.text + '\t')
9.    out.write('\n')

Explanation of code

  1. Creates a Document object using the WORD file.
  2. Document has a built-in function that gets tables in a list and here we are getting the one and only table the Word document has.
  3. Iterates through the text in each cell of each row to write it to the output text file.
