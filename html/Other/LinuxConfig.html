Installing Virtual Machine

Follow these step-by-step instructions to install the virtual box to your machine. Stick with the recommendations of how much memory to give to the VM depending on how much RAM your machine has. For storage at slide 10, you do not have to give 32GB for the virtual hard disk. GEMINI tools and software packages doesn't require a bunch of space, so you could give around 20 GB.

Installing GEMINI Tools

Pycharm

Pycharm is a useful IDE tool for editing, testing, and running python. Here are the steps to install Pycharm. Once it is done, you can clone the DahlgrenTest repository to Pycharm.

QT Creator

QT Creator is an IDE for C++, JavaScript, and QML that simplifies GUI application development. You can install this tool with the following yum command: sudo yum install qt-creator. Note you need to install these following libraries before being able to create a GEMINI workspace in QT Creator using sudo yum install:

  • epel-release
  • glibc.i686
  • gcc-c++.x86_64
  • nss-util.i686
  • glibc-devel.i686
  • mesa-libGL-devel.i686
  • qt-devel.i686
  • rpm-build.x86_64
  • libreoffice-writer.x86_64

Git

Unfortunately, linux does not support GitHub desktop, so you have to install git with sudo yum install git. Afterwards, clone the repository with git clone url where you get the url from the "Code" drop down button on the "<> Code" tab in the repos. To switch to a specific branch, use git checkout 'existing_branch'.
