Gaining access to the DahlgrenTest repository

These instructions are intended to help a beginner to gain access to the DahlgrenTest repository.

  • Go to GitHub.com and sign up for an account.
    • You need to create an handle for your account. Think of this like the call sign for a jet pilot.
    • PLEASE consider that this handle might appear in an award write up one day. You want to be proud of that handle even if it expresses your personality. HeavyLifter might be okay. Pussycat69 is not.
    • This handle is also the base of any repository you create for yourself, your resume, your portfolio, school work, etc.
  • Use a permanent email account that you will have access to in the future.
  • Go to your email and confirm your GitHub account.
  • Send an email to Brian Dillon requesting access to the DahlgrenTest repo. Include your handle so that you can be granted access.
  • You will likely need to check the email associated with your GitHub account that was invited to the repo.
  • Once you have gained access, you can follow this link to the DahlgrenTest repository.
  • From your GitHub page, download the desktop application that will let your push/pull code from the repository.
  • Refer to this guide on how to download GitHub Desktop
