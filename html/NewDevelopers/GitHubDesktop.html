Downloading GitHub Desktop and cloning the DahlgrenTest repository

These instructions will show you how to download GitHub Desktop so that you can contribute to the DahlgrenTest repository.

  • Go to GitHub Desktop and procede to download and install GitHub Desktop.
  • After installing GitHub Desktop, log in and navigate to the top left corner where it says file.
  • Then, select "Clone Repository" and in the window that pops-up, select "bsdillon/DahlgrenTest" and click "clone".
  • You now have all of the DahlgrenTest files in the directory "Documents/GitHub/DahlgrenTest" on your local machine.
  • In the future when you change branches in the DahlgrenTest repo, the files in the "Documents/GitHub/DahlgrenTest" directory will be swapped with whichever branch you are working on.
