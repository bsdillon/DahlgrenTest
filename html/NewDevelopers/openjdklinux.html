Installing OpenJDK on Linux Virtual Machine

Terminal Emulator

  1. Open the Terminal Emulator in the Linux Virtual Machine.



  3. Type sudo apt install openjdk-16-jdk into Terminal Emulator



  4. Type Y to continue. It may take a few more seconds to install.

  5. Type FILENAME.java to create a file. This will also open up the file editor.

  6. Type java FILENAME.java to run program.
