Installing OpenJDK on Linux Virtual Machine

Terminal Emulator

  1. Open the Terminal Emulator in the Linux Virtual Machine.



  2. Type sudo apt install openjdk-16-jdk into Terminal Emulator



  3. Type Y and press enter to continue. It may take a few more seconds to install.




  4. Type nano FILENAME.java to create a file. This will also open up the file editor.
    We will be creating a file with the name 'helloworld' so our command will be nano helloworld.java




  5. In GNU nano we will write a simple Hello World program. Here is what we will type:

    public class HelloWorld { public static void main(String[] args) { // Prints "Hello, World" in the terminal window. System.out.println("Hello, World"); } }





  6. Type java FILENAME.java to run program.
