Setting up Mathematica on your VM

Terminal Emulator

  1. Download and istall Mathematica on your Virtual Machine using this link

  2. Open the Terminal Emulator in the Linux Virtual Machine.





  3. Type cd /home/azureuser/Downloads to navigate to dowloads.

  4. Type ls to view the file name of the mathematica download.

  5. Type sudo bash [Mathematica file name] to navigate.
    Go through and follow the remaining prompts. Installing into the directory can take a while.

  6. Once finished, type Mathematica and press enter to continue. It will take a few seconds to load the first time.

  7. Now that we have downloaded Mathematica we will get it activated using MathLM which is on Dr.Tipton's server. Click "other ways to activate".



  8. Click "connect to a network license server" and use the public address 20.98.165.144.

  9. The program has been installed.Type mathematica in the terminal to access the program.
